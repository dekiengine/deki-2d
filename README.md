# Deki 2D

2D rendering components for the Deki Engine: sprites, text, animations, gradients, buttons, scrolling, rollers, clipping, and sorting groups.

Part of the [Deki Engine](https://github.com/ATeam-Dekiru/deki-engine) module ecosystem.

## Installation

Install via the Module Manager inside the Deki Editor.

## Dependencies

| Dependency | Type |
|---|---|
| `deki-rendering` | Deki module |
| `deki-input` | Deki module |
| `deki-editor` | Deki editor DLL |
| `freetype` | External (FreeType License) |
